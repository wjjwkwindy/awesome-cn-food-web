## 特性

- 极速搜索（Fuse.js）
- 网页版直接访问
- 人性化的地图界面（mapbox-gl）
- 调起三方地图进行导航（高德、百度地图）
- 当前位置到店铺的距离
- Vue3 + Vite

> 💡 店铺数据仅供参考，店铺评论为本人主观意见，如有侵权请联系我删除。
